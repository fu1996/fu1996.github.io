被调试的代码库​

redux-saga-mate

  1. 拉取对应的代码库，并安装相关依赖

  2. 因为要调试 第三方库 所以要时刻监听 代码库的修改

  3. 修改 packgae.json 文件


  4. -    "build": "rollup -c",
    +    "build": "rollup -c --watch --sourcemap",

  5. 将该代码库 代理到本地

  6. npm link

  7. 在项目中使用 该 代码库

  8. npm link redux-saga-mate

9. 随便在代码库中 写个 console 看控制台
